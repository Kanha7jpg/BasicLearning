# Space Invaders (Java)

This is a Space Invaders game made with Java Swing.

## How to run

1. Open terminal in this folder.
2. Compile:

```bash
javac Space_Invaders.java
```

3. Run:

```bash
java Space_Invaders
```

## Controls

- `Left Arrow` or `A` - move left
- `Right Arrow` or `D` - move right
- `Space` or `Up Arrow` - shoot
- `P` - pause/resume
- `R` - restart
- `N` - next level after win

## Goal

- Destroy all aliens.
- Do not lose all your lives.

## Requirement

- Java JDK 8 or newer
