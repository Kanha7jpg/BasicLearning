# TicTacToe (Java)

This is a 2-player TicTacToe game in Java.

## How to run

1. Open terminal in this folder.
2. Compile the code:

```bash
javac TicTacToe.java
```

3. Start the game:

```bash
java TicTacToe
```

## How to play

- Player 1 uses `X`
- Player 2 uses `O`
- Type a number from `1` to `9` to place your mark
- First player with 3 in a row wins

Board map:

```text
 1 | 2 | 3
---+---+---
 4 | 5 | 6
---+---+---
 7 | 8 | 9
```

## Requirement

- Java JDK 8 or newer
